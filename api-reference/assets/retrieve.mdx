Retrieve the metadata for a previously uploaded asset. Use this endpoint to look up an asset's CDN URL, verify its file size and content type, or confirm that an upload completed successfully. The response includes all fields from the original [Asset object](/api-reference/assets/upload#asset-object).

## Endpoint

```
GET https://api.techblazing.com/v1/assets/{id}
```

## Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the asset to retrieve. Asset IDs are prefixed with `ast_`.
</ParamField>

## Response

Returns an Asset object on success.

<ResponseField name="id" type="string">
  Unique identifier for the asset, prefixed with `ast_`.
</ResponseField>

<ResponseField name="object" type="string">
  The object type. Always `"asset"`.
</ResponseField>

<ResponseField name="filename" type="string">
  The stored filename of the uploaded asset, either the original name from the upload or the custom filename you provided.
</ResponseField>

<ResponseField name="content_type" type="string">
  The MIME type of the uploaded file. One of `image/jpeg`, `image/png`, or `image/webp`.
</ResponseField>

<ResponseField name="size" type="integer">
  The file size in bytes.
</ResponseField>

<ResponseField name="url" type="string">
  The CDN URL where the asset file can be accessed. This URL is stable for the lifetime of the asset.
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp recording when the asset was uploaded to TechBlazing.
</ResponseField>

```json Example response
{
  "id": "ast_01H9K7XVMZQ3P4R5T6Y8W",
  "object": "asset",
  "filename": "product-shot.jpg",
  "content_type": "image/jpeg",
  "size": 2048576,
  "url": "https://cdn.techblazing.com/assets/ast_01H9K7XVMZQ3P4R5T6Y8W.jpg",
  "created_at": "2024-07-01T14:20:00Z"
}
```

## Code examples

<CodeGroup>

```bash curl
curl --request GET \
  --url https://api.techblazing.com/v1/assets/ast_01H9K7XVMZQ3P4R5T6Y8W \
  --header "Authorization: Bearer $TECHBLAZING_API_KEY"
```

```javascript Node.js
import fetch from "node-fetch";

const assetId = "ast_01H9K7XVMZQ3P4R5T6Y8W";

const response = await fetch(
  `https://api.techblazing.com/v1/assets/${assetId}`,
  {
    headers: {
      Authorization: `Bearer ${process.env.TECHBLAZING_API_KEY}`,
    },
  }
);

const asset = await response.json();
console.log(asset.url); // https://cdn.techblazing.com/assets/ast_01H9K7XVMZQ3P4R5T6Y8W.jpg
```

```python Python
import os
import requests

asset_id = "ast_01H9K7XVMZQ3P4R5T6Y8W"

response = requests.get(
    f"https://api.techblazing.com/v1/assets/{asset_id}",
    headers={"Authorization": f"Bearer {os.environ['TECHBLAZING_API_KEY']}"},
)

asset = response.json()
print(asset["url"])  # https://cdn.techblazing.com/assets/ast_01H9K7XVMZQ3P4R5T6Y8W.jpg
```
